Corium
======

Small statically linked coreutils for Linux (x86_64).

Utilities implemented so far:
* `arch`
* `basename`
* `cat`
* `chroot`
* `chvt`
* `clear`
* `cmp`
* `dirname`
* `dmesg`
* `echo`
* `env`
* `false`
* `hostname`
* `mesg`
* `nice`
* `nohup`
* `printenv`
* `pwd`
* `renice`
* `reset`
* `rmdir`
* `sleep`
* `sync`
* `tee`
* `true`
* `uname`
* `yes`
